# DSA Preparation — Full Overview

Part of a placement-prep journey running from mid-July 2026 toward placement season, alongside Core (Python → Stats → ML → DL → NLP → GenAI), Quantitative Aptitude, Verbal/Logical Reasoning, and English.

## Goal

Build genuine, retained understanding of Data Structures & Algorithms for placement interviews — prioritizing depth over rushing to a deadline, since the point is to internalize concepts well enough to reason through unfamiliar problems, not just finish a checklist.

## Approach

- **Bootcamp phase (Jul 15–31):** Complexity Analysis, alongside finishing Python/Stats/EDA in the Core track.
- **From August onward:** self-paced, sequential — each topic gets as long as it genuinely needs before moving to the next. Loose checkpoints are used as sanity checks (e.g. "is this taking 3x longer than expected?"), not hard deadlines.
- Daily study happens in an evening block (~7:00–10:00 PM on college days), with DSA typically getting a dedicated ~1 hour slot after Core.

## Topic Sequence

1. **Complexity Analysis** — Time complexity, Space complexity, Big-O/Big-Ω/Θ/little-o/little-ω, nested loop analysis, sorting algorithm complexities (Bubble, Selection, Insertion), recursion & recurrence relations (Master's Theorem intuition, Akra-Bazzi — flagged as beyond typical placement scope but useful for depth)
2. **Arrays & Strings**
3. **Recursion**
4. **Linked List → Stack → Queue** (in this order)
5. **Hashing + Sorting**
6. **Trees**
7. **Graphs** — BFS/DFS basics
8. **Full Revision** — timed practice across all topics before placement mode

## Reference Material

- *Data Structures and Algorithms in Python* (Goodrich, Tamassia, Goldwasser) — Chapter 3, "Algorithm Analysis" (pages 111–141 printed / 133–163 in the PDF used), covering experimental studies, the seven fundamental functions, and Big-O notation
- *Data Structures and Algorithms in Python: An Indian Adaptation* — extends the above with Big-Omega, Theta, little-o, and little-omega notation (Section 3.4.1–3.4.6), not covered in the original US edition
- Personal handwritten notes covering the same ground plus recurrence-solving methods
